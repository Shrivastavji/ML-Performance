# 🎓 Student Performance Prediction using Machine Learning

## 📌 Project Overview

This project aims to predict whether a student will pass or fail in mathematics based on their reading and writing scores using machine learning models.

The goal is to demonstrate a complete machine learning workflow including data analysis, model building, evaluation, and optimization.

---

## 📊 Dataset

* Source: Students Performance Dataset
* Features used:

  * Reading Score
  * Writing Score
* Target:

  * Math Result (Pass/Fail)

---

## ⚙️ Technologies Used

* Python
* Pandas & NumPy
* Matplotlib & Seaborn
* Scikit-learn

---

## 🔍 Exploratory Data Analysis (EDA)

* Checked dataset structure and summary statistics
* Visualized distributions of scores
* Analyzed correlations between features
* Observed relationships between reading, writing, and math scores

---

## 🤖 Models Implemented

* Logistic Regression
* Decision Tree Classifier
* Random Forest Classifier (with Hyperparameter Tuning)

---

## 📈 Model Evaluation

Models were evaluated using:

* Accuracy Score
* Confusion Matrix
* Classification Report (Precision, Recall, F1-score)

---

## 🏆 Results

* Random Forest achieved the best performance
* Hyperparameter tuning improved model accuracy
* Writing score showed strong influence on predictions

---

## 📊 Feature Importance

Feature importance analysis revealed that:

* Writing score slightly contributes more than reading score
* Both features are strong predictors of math performance

---

## 🧠 Key Insights

* Students with higher reading and writing scores tend to perform better in mathematics
* Balanced performance across subjects improves prediction accuracy
* Ensemble models like Random Forest perform better than individual models
